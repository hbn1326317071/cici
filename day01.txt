WEB=浏览器+WEB服务器+HTTP通讯协议
  1 浏览器客户端（客户端）
    向服务器发送HTTP请求，接收并且处理服务
    器的响应结果。响应结果中经常是HTML
    在浏览器中显示HTML页面。
  2 WEB服务器(服务器)
    处理浏览器的请求，将处理结果发送浏览器
    处理结果经常是HTML。
  3 HTTP通讯协议
    浏览器和服务器之间的通讯协议。

HTML 是什么
  固定标签（标记，tag）的XML

XML
  标签：分为3种
  <标签名> 开始标签
  </标签名> 结束标签
  <标签名/> 自结束标签，相当于一对空标签
    <标签名/> == <标签名></标签名>
    自结束标签是空标签的简洁写法。
  元素：开始标签 + 内容 + 结束标签
  <name>Tom</name>
   内容: 文本，子元素，文本+元素混合
  属性：在开始标签上定义的 属性="值"
    属性无顺序
    属性必须使用引号
    属性不能重复
    只能在开始标签上定义

XML 格式良好要求
  有唯一的根元素
  标签必须配对使用，自结束是一对标签
  标签要正确嵌套
  
  XML 是严格区分大小写的。

XML 的可扩展性
  1 标签名可以任意定义（扩展）
  2 属性可以扩展
  3 内容可以扩展

HTML
  1 语法与XML基本相同（99%）
    HTML 不区别大小写，等
  2 标签固定
    根元素必须是<html>
    html 元素的内容只能是 <head> <body>
  3 属性也规定好了。
  4 有些元素的嵌套关系也规定了
